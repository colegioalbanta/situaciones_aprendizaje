Guía de Uso Completa y Configuración de API

🔑 ¿Cómo crear gratis tu API Key de Google Gemini?

La API de Google Gemini es el motor de Inteligencia Artificial recomendado para esta herramienta debido a su alta velocidad y gran precisión curricular. Sigue estos pasos para obtener una clave gratuita en menos de 2 minutos:

  1. Accede al portal oficial de desarrolladores de Google: Google AI Studio e inicia sesión con tu cuenta de Google (Gmail).
  2. Haz clic en el botón azul de la esquina superior izquierda que dice "Get API Key" (Obtener clave de API).
  3. Pulsa sobre "Create API Key" (Crear clave de API).
  4. Selecciona la opción "Create API Key in new project" para generar tu clave exclusiva de manera segura.
  5. Copia la clave generada (empieza por AIzaSy...) y pégala en el panel izquierdo de esta aplicación.

Pasos para Generar una Situación de Aprendizaje (SA)

1 Introduce la API Key

Pega tu clave en el campo "API Key Personal". Asegúrate de tener seleccionado el proveedor correcto (Gemini u OpenAI) en el menú superior del panel de conexión.

2 Elige los Parámetros Curriculares

Selecciona la etapa, el curso y la materia correspondientes. Los menús desplegables se adaptan dinámicamente según las directrices LOMLOE. Configura las sesiones e indica el núcleo temático (por ejemplo: "El ciclo del agua" o "Energías renovables").

3 Opciones DÚA e Inclusión (Opcional)

Mantén activa la casilla de verificación DÚA para que la Inteligencia Artificial incorpore medidas de inclusión y andamiaje cognitivo en cada sesión redactada. También puedes adjuntar un PDF de referencia como material de apoyo.

4 Generación y Exportación Directa

Haz clic en el botón "Generar Situación de Aprendizaje". Tras unos segundos, la previsualización se actualizará con toda la estructura de sesiones y rúbricas. Finalmente, haz clic en el botón verde de la zona superior derecha para descargar tu documento oficial en PDF sin que se corte el contenido.

💡 Nota del Sistema: Si no tienes conexión a internet o no dispones de una clave API activa en este momento, puedes desmarcar la casilla de IA para generar una estructura técnica estándar al instante de manera offline.
